# Security Policy

AI IME 会处理键盘输入、模型密钥和输入法配置。请不要在公开 issue 中粘贴真实键盘日志、数据库或 API Key。

## 支持版本

当前项目处于 Alpha，只有 `main` 分支接受安全修复。

## 报告安全问题

如果发现安全问题，请创建一个不含敏感数据的 issue，说明：

- 影响范围。
- 复现步骤。
- 是否涉及键盘日志、API Key、Rime 配置或远程模型上传。

如果问题包含真实密钥或个人输入内容，请先脱敏。

## 项目默认安全边界

- `.env` 被 `.gitignore` 忽略。
- UI 只展示脱敏 API Key。
- 云端/中转模型默认不接收完整键盘日志。
- Rime 部署会备份被覆盖文件。
